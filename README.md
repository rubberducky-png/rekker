rekker — Personal Portfolio Website
rekker is my personal portfolio website, built to showcase my projects, skills, and creative work through a clean and modern interface. The site focuses on smooth UI interactions, intentional design, and subtle animations that create a polished experience without unnecessary complexity. It includes a fixed title bar with smooth auto‑scroll navigation, section‑specific background styles, and an interactive projects area powered by a custom dot‑grid canvas animation.

Features
Minimal, responsive layout designed around clarity, spacing, and typography

Fixed navigation bar with smooth scrolling to each section

Unique background styling for each section, including ambient gradients, geometric patterns, and canvas animations

Interactive Projects section with horizontal scrolling cards, hover animations, and tech stack tags

Custom dot‑grid canvas animation running behind the Projects section

Clean content blocks with consistent padding and layout structure

Tech Stack
HTML

CSS

JavaScript

The site uses custom canvas animations, responsive design techniques, and minimalistic typography to maintain a cohesive visual identity.
